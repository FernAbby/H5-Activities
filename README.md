# H5-Activities
存放部分活动页面,包含投票,问卷,大转盘,砍价,刮刮卡,摇一摇,优惠券,意见收集活动......

一部分是自己写过活动，一部分是用到网络素材（如原作者看到介意分享，可联系本人删除）

## 项目新运行

```
git clone https://github.com/FernAbby/activities.git

cd activities

npm i

npm start 

```
